# Substack cross-posts

**How to publish each file:**
1. Substack → **New post** → **Text**.
2. Paste the `.md` body. Substack's editor renders Markdown on paste (headings, lists, bold, links, code, tables)
   and **auto-embeds images** from the `https://crlab.ca/...` URLs — give them a second to load before publishing.
3. Set the post **subtitle** from the first italic line, then delete that line if you prefer.
4. Substack has no external-canonical, so these are written as the *newsletter edition* (own intro + a
   "full version on crlab.ca" link) rather than a byte-identical duplicate — keeps you clear of duplicate-content issues.
5. Optionally group posts into Substack **sections** (e.g. "AI explainers", "Careers", "Security research").
